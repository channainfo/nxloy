## Domain Purpose

Manage influencer network, viral challenges, and social graph for the NxLoy platform.

## Core Concepts

**Influencer Network**: AI-powered micro-influencer discovery (10K-50K followers), audience matching, automated outreach

**Viral Challenges**: TikTok/Instagram challenges with AI trend detection, team competitions, real-time leaderboards

**Social Graph**: Customer network analysis, super referrer identification, influence scoring

## Aggregates

1. **Influencer** - Partnership management with ROI tracking
2. **Challenge** - Viral challenge with participation tracking
3. **SocialGraph** - Network size and influence scoring
